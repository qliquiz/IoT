# car_calibration

Скетч [car_calibration](car_calibration.cpp) — это интерактивная калибровка 2‑моторной машинки по Bluetooth (через
`SoftwareSerial`), с сохранением параметров в EEPROM.

## Что делает скетч

Программа по шагам настраивает:

1. **Направления моторов** для действий:
    - движение вперёд,
    - назад,
    - разворот влево,
    - разворот вправо.
2. **Баланс скоростей** левого/правого колеса.
3. **Время поворота** на 90° / 180° / 270° / 360°.

После калибровки параметры сохраняются в EEPROM и при следующем старте автоматически загружаются.

---

## Пины и интерфейсы

- Моторы:
    - `DIR_RIGHT = 4`
    - `SPEED_RIGHT = 5` (PWM)
    - `SPEED_LEFT = 6` (PWM)
    - `DIR_LEFT = 7`
- Bluetooth: `SoftwareSerial bt(2, 3)`
- Лог в USB Serial: `57600`

---

## Режимы работы

В коде есть 4 режима (`Mode`):

- `DIR_CAL` — калибровка направлений
- `SPEED_BAL` — балансировка скоростей
- `TURN_TIME` — настройка времени поворотов
- `RUN` — обычное управление машинкой

### 1) DIR_CAL

Подбор правильной пары направлений для каждого действия (`FORWARD`, `BACKWARD`, `ROTATE_LEFT`, `ROTATE_RIGHT`).

- `F` — запустить моторы с текущей комбинацией
- `B` — стоп
- `X` — следующая комбинация направления (4 варианта)
- `S` — подтвердить комбинацию для текущего действия
- `R` — пропустить текущее действие
- `C` — вернуться к предыдущему действию
- `T` — перейти в `SPEED_BAL`

### 2) SPEED_BAL

Подстройка PWM для каждого колеса.

- `F` — тест движения вперёд
- `B` — стоп
- `X` — выбрать редактируемое колесо (Left/Right)
- `R` — увеличить скорость выбранного колеса на +5
- `L` — уменьшить скорость выбранного колеса на -5
- `C` — сброс обоих скоростей к `BASE_SPEED`
- `T` — перейти в `TURN_TIME`

### 3) TURN_TIME

Калибровка длительности разворота.

- `F` — тестовый поворот влево (`ACT_ROTL`) на текущее время
- `R` — +50 мс
- `L` — -50 мс
- `S` — подтвердить текущий угол и перейти к следующему
- `X` — сброс времени текущего угла к дефолту
- `C` — вернуться к предыдущему углу
- `T` — сохранить параметры в EEPROM и перейти в `RUN`

### 4) RUN

Обычное управление после калибровки.

- `F` — вперёд
- `B` — назад
- `L` — влево
- `R` — вправо
- `S` — стоп
- `X` — сбросить признак калибровки в EEPROM и вернуться в `DIR_CAL`

---

## EEPROM: что сохраняется

По адресам EEPROM сохраняются:

- направления для 4 действий (`ACT_FWD`, `ACT_BWD`, `ACT_ROTL`, `ACT_ROTR`),
- скорости `speed_left`, `speed_right`,
- массив времен поворота `turn_ms[4]` (90/180/270/360),
- байт валидности `EEPROM_MAGIC = 0xAB`.

При старте:

- если `ADDR_VALID == 0xAB` → загрузка калибровки и вход в `RUN`,
- иначе старт с `DIR_CAL`.

---

## Ключевые значения по умолчанию

- `BASE_SPEED = 200`
- `DEFAULT_T90  = 400`
- `DEFAULT_T180 = 800`
- `DEFAULT_T270 = 1200`
- `DEFAULT_T360 = 1600`

---

## Краткий сценарий использования

1. Подключиться по Bluetooth и отправлять символы-команды (`F/B/L/R/S/X/C/T`).
2. Пройти `DIR_CAL` → `SPEED_BAL` → `TURN_TIME`.
3. В `TURN_TIME` нажать `T` для сохранения.
4. После перезагрузки машинка сразу стартует в `RUN` с сохранёнными параметрами.
