# SmartCar Configurator

Репозиторий содержит систему интерактивной настройки двухмоторной робо-платформы. Прошивка позволяет откалибровать
физические параметры моторов через Bluetooth-терминал и сохранить конфигурацию в энергонезависимую память (EEPROM).

## Основные возможности

Система автоматизирует три этапа подготовки робота:

1. **Коррекция полярности:** Определение правильных направлений вращения колес для каждого типа движения.
2. **Синхронизация тяги:** Выравнивание скорости левого и правого моторов для движения по прямой.
3. **Хронометраж маневров:** Точная настройка времени работы двигателей для поворотов на фиксированные углы.

## Аппаратное обеспечение

### Схема подключения

| Компонент                  | Пин Arduino | Тип сигнала |
|:---------------------------|:-----------:|:------------|
| Направление (Правый мотор) |      4      | Digital     |
| Скорость (Правый мотор)    |      5      | PWM         |
| Скорость (Левый мотор)     |      6      | PWM         |
| Направление (Левый мотор)  |      7      | Digital     |
| Bluetooth RX               |      2      | Soft Serial |
| Bluetooth TX               |      3      | Soft Serial |

**Параметры связи:**

* USB Serial: `57600 baud`
* Bluetooth: `9600 baud`

## Интерфейс управления (Команды)

В системе реализовано 4 последовательных режима. Переключение между ними осуществляется по логической цепочке.

### 1. Режим направления (`DIR_CAL`)

Цель: добиться, чтобы робот корректно выполнял команды движения.

- `F` / `B` — Тест / Остановка.
- `X` — Перебор комбинаций фазировки (всего 4 варианта).
- `S` — Сохранить текущую комбинацию для активного действия.
- `R` / `C` — Навигация по списку действий (Вперед -> Назад -> Влево -> Вправо).
- `T` — Переход к следующему этапу (Скорость).

### 2. Балансировка мощности (`SPEED_BAL`)

Цель: устранить "занос" машинки при движении прямо.

- `F` / `B` — Тестовый заезд / Стоп.
- `X` — Переключение между левым и правым колесом.
- `R` / `L` — Изменение скорости выбранного мотора (+5 / -5).
- `C` — Сброс до базового значения (`200`).
- `T` — Переход к настройке времени.

### 3. Калибровка углов поворота (`TURN_TIME`)

Цель: настройка времени работы моторов для точных поворотов.

- `F` — Выполнить поворот на заданный угол.
- `R` / `L` — Коррекция длительности (+50 мс / -50 мс).
- `S` / `C` — Переключение между углами (90° / 180° / 270° / 360°).
- `X` — Восстановить стандартное время для текущего угла.
- `T` — Финализация: **запись в EEPROM** и переход в рабочий режим.

### 4. Рабочий режим (`RUN`)

Стандартное управление откалиброванным роботом.

- `F`, `B`, `L`, `R` — Движение.
- `S` — Экстренное торможение.
- `X` — Полный сброс настроек и возврат к первому этапу калибровки.

## Технические детали хранения (EEPROM)

Прошивка использует `EEPROM` для хранения следующих данных:

* Матрица состояний пинов `DIR` для 4-х типов движения.
* Значения PWM для левого и правого бортов.
* Таблица задержек для разворотов (двухбайтовые значения).
* Маркер валидности: байт `0xAB` по адресу `0`.

Если при включении маркер `0xAB` не обнаружен, устройство принудительно запускает мастер настройки.

## Параметры "из коробки"

- Базовая скорость: `200`
- Тайминги (мс): `400` (90°), `800` (180°), `1200` (270°), `1600` (360°).

---

## Алгоритм первого запуска

1. Подключитесь к Bluetooth-модулю через любое терминальное приложение.
2. Последовательно пройдите этапы `DIR_CAL`, `SPEED_BAL` и `TURN_TIME`.
3. Нажмите `T` в последнем режиме, чтобы сохранить данные.
4. Теперь робот готов к эксплуатации и будет сохранять настройки даже после выключения питания.
